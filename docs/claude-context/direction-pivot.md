# 方向调整：QuantEvo → 智投助手

**调整日期**: 2026-06-01
**决策人**: 用户

## 调整前
- 定位：面向专业量化策略开发者的"策略生态健康度评估系统"
- 核心用户：量化策略管理者、算法交易员
- 核心功能：DNA测序、系统发育、代谢分析、寿命预测

## 调整后
- 定位：面向投资小白的"量化投资快速上手工具"
- 核心用户：投资小白、想入门量化投资的个人投资者
- 核心功能：投资画像 → 经济周期判断 → 智能组合推荐 → 策略回测验证 → 寿命监控预警

## 新增核心模块
1. **投资画像系统**：资金规模、风险偏好、流动性需求 → 画像标签 + 建议配置
2. **经济周期引擎**：宏观指标采集 → 周期阶段判断（复苏/过热/滞胀/衰退）→ 周期-资产映射
3. **智能组合构建器**：画像 + 周期 → 战略配置 + 战术配置 → 策略流自动生成
4. **策略推荐引擎**：画像 × 周期 × 策略库 → 个性化策略推荐
5. **策略寿命监控中心**：复用生态系统，聚焦用户策略的健康预警

## 资产范围扩展
- 原有：A股个股
- 新增：ETF、公募基金、债券（可转债/国债）、期货
- 排除：港股、美股（暂不支持）

## 数据源确认（2026-06-01验证）
- akshare 1.18.55 支持所有目标资产类别的数据获取
- ETF：`fund_etf_hist_em` 日频OHLCV
- 基金：`fund_open_fund_daily_em` 日频净值
- 债券：`bond_zh_cov` 可转债数据
- 期货：`futures_zh_daily_sina` 主力连续合约日频数据

## 回测适配策略
- ETF：直接可用（交易规则同股票）
- 基金：仅配置推荐，不参与回测（日频净值，申赎延迟）
- 债券：简化回测（利息+价格波动）
- 期货：简化模拟（先支持做多，忽略保证金/合约换月）

## 用户系统
- 简单手动切换（非登录注册体系）
- 所有数据按 user_id 隔离
- 前端顶部导航栏提供用户切换下拉框

## 策略来源
- 系统智能生成（简单策略、复杂策略、GitHub热门开源策略）
- 用户策略回测效果优秀者自动入池
- 开源公益项目，无商业化

## 保留功能包装
- DNA测序 → "策略体检报告"
- 代谢率 → "数据依赖度"
- 寿命预测 → "策略有效期"
- 策略工坊 → "高级模式"（降低入口优先级）

## 实施路线
- Phase A（2-3周）：用户系统 + 投资画像 + 首页重构
- Phase B（2-3周）：经济周期 + 组合构建
- Phase C（2-3周）：策略推荐 + 寿命监控
- Phase D（1-2周）：体验打磨 + 新手引导

## 合规
- 所有推荐需 prominently 展示"仅供参考，不构成投资建议"
- 开源公益项目定位

## Why
原方向面向专业量化开发者，市场小众且竞争已很激烈（聚宽、JoinQuant等）。转向投资小白后：
1. 目标用户扩大100倍
2. 自然语言解析 + 流水线编辑器天然适合降低门槛
3. 生态系统模块（寿命/健康）可转化为差异化卖点——"不只是推荐，还告诉你策略能活多久"
4. 开源公益定位降低运营压力

## How to apply
- 所有新功能开发需通过"小白视角"验收：术语是否通俗、操作是否三步内完成、是否有明确引导
- 专业术语保留但增加通俗解释（tooltip/帮助文档）
- 首页必须实现双状态：无画像时强制引导，有画像时展示个性化仪表盘
