## 五大集成方案

### 方案一：回测中心的"生态预审"
- 创建回测前调用 phylogeny 检查同质化风险，高相似度时弹窗警告
- 根据 metabolic_rate 智能推荐回测数据粒度（日频/周频）
- 回测完成后将 metrics 归档到 strategy_dna，验证寿命预测准确性

### 方案二：模拟盘的"寿命倒计时"
- 模拟盘列表显示剩余寿命（从 lifespan_months 换算）
- 连续 N 天收益低于 aging_velocity 预期阈值时发出衰竭预警
- 停止模拟盘时弹出"死亡原因"选择器，归档到策略墓地

### 方案三：策略工坊的"基因实时预览"
- 代码编辑器旁添加实时基因预览侧边栏（debounce 3 秒）
- 保存时检查修改后代码是否使相似度突破 80%，确认弹窗

### 方案四：生态仪表盘的操作闭环
- 需关注/寿命预警/策略墓地面板中，每个策略添加操作按钮组（编辑→策略工坊、回测→回测中心、模拟盘→模拟盘监控）
- 顶部导航栏添加生态指示灯（濒危策略时红色圆点）

### 方案五：全局策略列表的生态徽章
- 回测中心/模拟盘创建弹窗的策略选择器中显示健康度圆点、家族标签、同质化警告
- 策略工坊左侧列表显示迷你健康度条

## 实施顺序

1. 方案四（操作闭环）— 纯前端，改动最小，体验最直接
2. 方案五（全局徽章）— 纯前端，扩展生态数据展示范围
3. 方案一（回测生态预审）— 需要后端改模型（归档 backtest metrics）
4. 方案二（模拟盘寿命倒计时）— 需要后端改模型（模拟盘状态关联）
5. 方案三（基因实时预览）— 可选，改动最大
