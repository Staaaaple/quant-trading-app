# 策略自动生成模块设计决策

**日期**: 2026-06-01
**相关决策**: [[direction-pivot]]

## 一、方案演进路线：先B后A

### Phase B（经济周期+组合构建期）：改进版RAG
- RAG检索增加硬过滤（风险等级、适合周期）
- LLM只输出设计文档（JSON），不生成代码
- 代码由确定性模板引擎组装
- 迭代优化从"改代码"变为"调参数"
- 增加Walk-Forward验证

### Phase C（策略推荐+寿命监控期）：迁移到Hybrid引擎
- 模板参数化（60%）+ 规则组合（30%）+ GitHub策略（10%）
- 贝叶斯优化（Optuna/TPE）替代网格搜索
- Walk-Forward Analysis严格分离训练/验证/测试

## 二、Hybrid引擎架构

```
Layer 1: 知识沉淀
├── 策略代码库（~200个经过验证的策略）
├── 论文结构化数据库
└── 市场状态-策略表现历史映射

Layer 2: 策略设计（确定性规则引擎）
├── RAG检索 → 获取当前周期下表现最好的策略家族
├── 规则引擎 → 根据用户风险等级选择主家族+辅家族
├── 加载参数化模板
└── 定义参数搜索空间（根据RAG论文结论动态确定）

Layer 3: 参数优化（贝叶斯优化）
├── 目标函数: 综合评分 = 夏普×0.25 + 卡玛×0.25 + 胜率×0.2 + 收益回撤比×0.2 + 泛化稳定性×0.1
├── 搜索算法: Optuna/TPE贝叶斯优化
├── Walk-Forward验证: 训练60% / 验证20% / 测试20%
└── 早停机制: 验证集连续3轮不提升则停止

Layer 4: 泛化验证（硬门槛）
├── 测试1: 10只不同类型股票 × 每段≥6个月 × 不同时间段 → 胜率皆>40%
├── 测试2: 样本外测试（最近6个月零参与训练）
├── 测试3: DNA测序健康度 > 60
├── 测试4: 寿命预测 > 6个月
├── 测试5: 与池内现有策略综合同质化风险 < 0.7
└── 只有通过全部5项测试才入池
```

## 三、策略池架构（七大策略家族）

策略池不局限于经典技术指标，涵盖**学术前沿策略**（机器学习、因子挖掘、统计套利等）。

### 家族1: 经典技术指标家族（~30个模板）
- 均线策略: 双均线、三均线、均线+MACD确认
- 通道策略: 布林带、唐奇安通道、肯特纳通道
- 振荡器策略: RSI超买超卖、KDJ、CCI
- 突破策略: 价格突破、成交量突破、波动率突破

### 家族2: 机器学习策略家族（~25个模板）
- 监督学习:
  - 随机森林: 多因子输入预测次日涨跌概率
  - XGBoost/LightGBM: 特征选择+梯度提升分类
  - SVM: 核函数分类市场状态
- 深度学习:
  - LSTM: 时序序列预测未来N日收益
  - Transformer: 自注意力机制捕捉长程依赖
  - CNN: 卷积处理K线图模式识别
- 集成学习:
  - 投票集成: 多模型加权投票决策
  - Stacking: 元学习器组合基模型

### 家族3: 因子挖掘策略家族（~30个模板）
- 传统多因子:
  - Fama-French五因子: 市值(SMB)、价值(HML)、动量(UMD)、质量(RMW)、低波动(CMA)
  - Barra CNE5: 行业因子+风格因子暴露控制
  - Carhart四因子: FF三因子 + 动量
- 另类因子:
  - 舆情情感: 新闻情绪得分→择时信号 (基于NLP情感分析)
  - 资金流: 北向资金、主力资金流向、大宗交易溢价
  - 分析师预期: 预期EPS偏离度、评级上调密度
  - 机构持仓: 季度持仓变动跟踪、国家队动向
  - 供应链因子: 上下游企业盈利传导
- 微观结构因子:
  - 订单流不平衡(OFI): 买单/卖单比例
  - 买卖压力: 盘口深度变化、委托簿斜率
  - 高频收益率偏度: 日内收益分布
  - 量价背离: 价格与成交量的非同步性
  - 换手率突变: 异常成交量检测

### 家族4: 统计套利策略家族（~15个模板）
- 配对交易:
  - 协整检验+均值回归: Engle-Granger两步法
  - 距离法配对: 最小距离选 Pair
- 跨期套利:
  - 期货近远月价差: 正套/反套
- 跨品种套利:
  - 产业链套利: 豆粕-豆油-棕榈油
  - 贵金属套利: 黄金-白银比价

### 家族5: 波动率交易家族（~15个模板）
- GARCH族模型: GARCH/EGARCH/GJR-GARCH预测波动率
- 波动率择时: 高波减仓、低波加仓
- 波动率期限结构: 近月/远月波动率价差
- 隐含波动率vs实际波动率: 做空/做多波动率溢价

### 家族6: 行为金融策略家族（~20个模板）
- 动量+反转复合:
  - 短期反转(1月) + 长期动量(12月)
  - 行业动量+个股反转交叉
- 情绪指标:
  - 恐慌贪婪指数合成
  - 融资余额变化率
  - 新开户数变化
- 拥挤度指标:
  - 散户情绪反向指标
  - 一致性预期偏离度

### 家族7: 事件驱动策略家族（~10个模板）
- 财报超预期: 净利润 surprise 动量
- 分红送转: 高送转预案公告效应
- 指数成分调整: 纳入/剔除效应
- 解禁预警: 限售股解禁前避险

### 策略池来源
| 来源 | 数量 | 方式 |
|------|------|------|
| 经典策略手册 | ~30 | 人工代码化 |
| 学术论文复现 | ~40 | 人工复现+适配akquant |
| GitHub热门开源 | ~50 | 手动精选+安全审核+语法迁移 |
| 平台社区优秀策略 | ~30 | 用户策略入池机制 |
| **总计** | **~150个活跃策略** | |

## 四、关键设计决策

### 1. 不追求"原理级全新逻辑"
- 参数创新: 贝叶斯优化自动搜索 ✅
- 组合创新: 跨家族模板组合 ✅
- 模板更新: 季度人工review+新增
- 前沿策略引入: 论文/GitHub人工筛选适配
- **Hybrid引擎边界: 在已有模板库内做"参数优化"和"跨家族组合"**

### 2. 三层架构应对不同用户场景
- Layer 1+2（预生成池+实时匹配）: 90%用户场景，响应<1秒
- Layer 3（定制生成）: 10%进阶用户，异步3-5分钟

### 3. GitHub策略获取方式
- 手动精选50个策略
- 来源: Backtrader官方示例(10) + VN.PY官方CTA(10) + 学术论文开源实现(15) + Kaggle/量化社区(15)
- 不接受自动爬虫（安全不可控）

## 四、入池标准（定稿）

| 指标 | 标准 |
|------|------|
| 测试股票 | 10只不同类型（跨行业/市值/风格） |
| 单段回测 | ≥ 6个月 |
| 测试段数 | ≥ 3个不同时间段 |
| 胜率 | 每只、每段皆 > 40% |
| 年化超额 | > 沪深300同期 + 2% |
| 最大回撤 | < 25% |
| 夏普比率 | > 0.5 |
| 卡玛比率 | > 0.5 |
| 样本外测试 | 最近6个月不参与训练/优化 |

## 五、同质化风险三维模型

```
综合同质化风险 = 0.35 × 代码同质化 + 0.35 × 收益同质化 + 0.3 × 因子同质化

代码同质化: 32维基因向量的余弦相似度
收益同质化: 历史回测日收益序列的相关系数
因子同质化: 对市值/价值/动量/波动率/质量因子的暴露重叠
```

## 六、动态寿命更新机制

```
动态寿命 = 基础寿命（静态） × 波动率加速因子 × 周期适配因子 × 拥挤度因子

波动率加速因子:
  当前波动率百分位 > 150% → 1.3
  当前波动率百分位 > 120% → 1.15
  当前波动率百分位 < 50% → 0.85

周期适配因子:
  当前周期 in 策略适应周期 → 1.0
  相邻周期 → 0.9
  不适配 → 0.7

拥挤度因子（sigmoid）:
  家族超额收益 > 10% → 0.75（表现好→资金涌入→未来拥挤）
  家族超额收益 > 5% → 0.9
  家族超额收益 < -5% → 1.1（表现差→资金流出→不拥挤）
```

**更新频率**: 每月1日自动运行
**预警阈值**: 老化速度>20%黄色预警、>40%红色预警
**数据存储**: strategy_lifespan_history表记录寿命轨迹

## 七、资产回测适配策略

| 资产 | 回测方式 | 说明 |
|------|---------|------|
| 个股 | akquant原生 | 日频OHLCV |
| ETF（含商品期货ETF） | akquant原生 | 交易规则同股票 |
| 可转债 | akquant原生 | 交易规则同股票 |
| 公募基金 | 独立fund_backtest_engine | 净值+申购费+赎回费+T+1确认+分红再投 |
| 期货合约 | 暂不开放直接交易 | 通过商品期货ETF间接配置 |

## 八、已知缺陷与应对

| 缺陷 | 应对方案 |
|------|---------|
| Hybrid引擎局部最优 | 多起点优化（5个初始点并行） |
| WFA数据损耗 | 要求原始数据至少5年 |
| 模板质量决定下限 | 模板人工审核+历史验证 |
| 冷启动问题 | 固定模板库兜底 |
| 无法生成原理级新逻辑 | 季度人工review更新模板库 |
