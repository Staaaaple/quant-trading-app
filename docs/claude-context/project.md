## 已完成

- **Phase 0 — DNA 测序引擎**：AST 解析提取四层基因（特征/信号/风控/执行），32 维基因向量，健康度评分，自动触发测序
- **Phase 0 — 种子策略库**：20 个预设策略（趋势/动量/均值回归/多因子/风控增强），种子脚本可重复运行
- **Phase 0 — 策略地图**：按家族分组卡片展示，支持导入策略
- **Phase 0 — 基因报告页**：四层基因标签、健康度/多样性、家族信息
- **Phase 1 — 系统发育引擎**：余弦相似度计算、Top 5 近亲、同质性风险、近亲警告、家族聚类
- **首页生态仪表盘**：双状态（无用户策略时展示内置地图+理想生态预览；有用户策略时展示真实 KPI + 图表）
- **用户手册**：重写为 QuantEvo 专用，10 个可折叠章节
- **Phase 2 — 代谢分析器**：信息代谢率、生态位宽度、代谢综合征检测

## 待办

- **Phase 3 — 寿命预测器**：综合代谢率+生态位宽度+同质化压力预测策略 edge 剩余寿命
- **Phase 4 — 生态仪表盘完整实现**：策略家族树可视化、代谢率趋势图、竞争压力雷达图、策略墓地

- **反参考**：避免医疗/日式密集 UI，无 glassmorphism/霓虹发光
