## 二、页面模块详细逻辑

### 页面1: 首页 (HomeView.vue)

**状态分支：**
```
用户访问 /
    │
    ├──→ [检查 localStorage 当前用户ID]
    │      │
    │      ├── 无用户ID ──→ 显示"空状态引导页"
    │      │                  │
    │      │                  ├── 大标题: "智投助手 — 你的AI投资顾问"
    │      │                  ├── 副标题: "3分钟完成画像，获取个性化投资策略"
    │      │                  ├── 三步流程图: 画像 → 诊断 → 配置
    │      │                  └── CTA按钮: "开始投资画像" → /profile
    │      │
    │      └── 有用户ID ──→ 显示"个人投资中心Dashboard"
    │                         │
    │                         ├── 顶部: 用户切换器 + 当前画像摘要
    │                         ├── 第一行卡片:
    │                         │   ├ 当前经济周期 (复苏/过热/滞胀/衰退)
    │                         │   ├ 市场环境评分 (-100~+100)
    │                         │   ├ 我的策略健康度 (健康/关注/濒危数量)
    │                         │   └ 寿命预警 (最近到期策略)
    │                         ├── 第二行: 我的投资组合概览
    │                         │   ├ 总资产、今日收益、累计收益
    │                         │   ├ 配置饼图 (股票/债券/基金/现金)
    │                         │   └ 收益曲线 (策略 vs 基准)
    │                         ├── 第三行: 快速操作
    │                         │   ├ [调整画像] → /profile
    │                         │   ├ [查看组合] → /portfolio
    │                         │   ├ [策略推荐] → /recommendation
    │                         │   └ [寿命监控] → /lifespan
    │                         └── 底部: 策略工坊快捷入口 (折叠)
```

**数据依赖：**
```
GET /api/v1/users/me                          → 用户信息
GET /api/v1/investor-profiles/me              → 投资画像
GET /api/v1/market-signals/latest             → 最新市场信号
GET /api/v1/portfolios/me/summary             → 组合概览
GET /api/v1/lifespan-monitor/alerts           → 寿命预警列表
```

### 页面2: 投资画像问卷 (InvestorProfileWizard.vue)

**问卷流程（6步）：**
```
Step 1: 基本信息
    ├── 资金规模（单选）: <5万 / 5-20万 / 20-100万 / >100万
    └── 投资经验（单选）: 无经验 / 1年内 / 1-3年 / 3年以上

Step 2: 风险偏好
    ├── 能承受的最大亏损（单选）: 5%以内 / 5-15% / 15-30% / 30%以上
    ├── 亏损时的心态（单选）: 立即止损 / 再观察 / 逢低加仓 / 无所谓
    └── 风险自评（滑动条）: 1-10分

Step 3: 投资目标
    ├── 投资期限（单选）: 1年内 / 1-3年 / 3-5年 / 5年以上
    ├── 预期年化收益（单选）: 跑赢通胀 / 5-8% / 8-15% / 15%以上
    └── 投资目标（多选）: 资产保值 / 稳健增值 / 财富增长 / 激进收益

Step 4: 流动性需求
    ├── 每月可能需要赎回比例（单选）: 0% / 10% / 25% / 50%
    └── 是否有大额支出计划（单选）: 无 / 1年内有 / 3年内有

Step 5: 资产偏好
    ├── 了解程度排序（拖拽）: 股票 / 基金 / 债券 / 黄金 / 期货
    └── 排斥资产（多选）: 无不排斥 / 股票 / 期货 / 都不排斥

Step 6: 确认与生成
    ├── 展示画像标签: "积极型长期投资者"
    ├── 展示建议配置饼图预览
    ├── [返回修改] / [确认生成]
    └── 确认后 → POST /api/v1/investor-profiles → 跳转Dashboard
```

**画像计算逻辑：**
```python
# 后端计算
risk_score = (
    最大亏损容忍度(40%) + 
    风险自评(30%) + 
    投资经验调整(20%) + 
    亏损心态(10%)
)

profile_type = {
    risk_score <= 3: "conservative" (保守型),
    risk_score <= 5: "balanced" (稳健型),
    risk_score <= 7: "aggressive" (积极型),
    risk_score > 7: "speculative" (激进型)
}

suggested_allocation = {
    "conservative": {"stock": 0.20, "bond": 0.50, "fund": 0.25, "cash": 0.05},
    "balanced":     {"stock": 0.40, "bond": 0.35, "fund": 0.20, "cash": 0.05},
    "aggressive":   {"stock": 0.60, "bond": 0.20, "fund": 0.15, "cash": 0.05},
    "speculative":  {"stock": 0.75, "bond": 0.10, "fund": 0.10, "cash": 0.05}
}
```

## 五、用户核心任务流程

### 任务1: 首次使用完成画像并获取推荐（目标: 3分钟内）

```
打开首页 → 点击"开始投资画像" → 完成6步问卷 → 查看画像结果
    → 点击"查看推荐组合" → 浏览组合配置 → 点击"策略推荐"
    → 浏览推荐策略列表 → 选择策略加入组合 → 点击"回测验证"
    → 查看回测结果 → 满意则"应用组合"
```

### 任务2: 定期查看策略健康并调整（目标: 1分钟内）

```
打开首页 → 查看Dashboard寿命预警卡片 → 点击红色预警
    → 进入寿命监控中心 → 查看濒危策略详情
    → 点击"寻找替代策略" → 浏览推荐替代方案
    → 选择替代策略 → 一键替换 → 回测验证新组合
```

### 任务3: 根据市场变化调整配置（目标: 2分钟内）

```
打开首页 → 查看市场信号卡片（发现周期变化）
    → 点击进入市场信号详情 → 确认周期判断
    → 返回首页 → 点击"调整组合"
    → 组合构建器自动建议新配置 → 对比新旧配置
    → 确认调整 → 回测验证 → 应用新组合
```
