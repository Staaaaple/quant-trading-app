## 四、核心任务流程

### 任务1: 首次使用（目标: 5分钟完成画像→看到组合→了解怎么操作）

```
打开首页 → 点击"开始画像" → 完成10题问卷 → 查看画像结果
    → 点击"查看完整组合" → 浏览组合配置（收益/风险/寿命）
    → 点击"应用组合" → 弹出教学引导（4步）
    → 完成教学 → 进入Dashboard
```

### 任务2: 日常跟进（目标: 1分钟了解今日操作）

```
打开首页Dashboard → 查看"今日操作"卡片
    ├── 有操作 → 查看买入/卖出信号 → 一键确认或忽略
    └── 无操作 → 查看"今日持有" → 快速退出
    
每周五: 查看周报推送 → 收益回顾 → 市场回顾 → 下周展望
```

### 任务3: 收到预警后处理（目标: 2分钟完成调仓决策）

```
收到推送: "策略A寿命到期，建议替换"
    → 点击推送进入Dashboard
    → 查看寿命监控卡片
    → 点击"查看替代方案"
    → 浏览3个替代策略（对比夏普/回撤/寿命）
    → 选择替代策略 → 确认调仓
    → 查看调仓后的新组合回测结果
    → 应用调仓
```

### 任务4: 根据市场变化主动调整（目标: 3分钟）

```
Dashboard发现市场周期变化（如复苏→过热）
    → 点击市场信号卡片
    → 查看周期判断依据
    → 点击"调整组合"
    → 组合构建器自动建议新配置
    → 对比新旧配置差异
    → 确认调整 → 回测验证 → 应用
```
